# Tesla EV Dashboard - Washington State

This R Flexdashboard project analyzes Tesla's electric vehicle market performance across Washington State using public registration data. It was built as part of mycompletion of the **R Programming for Data Analysts** course with Analyst Builder.

Tools Used 
- R (tidyverse, dplyr, ggplot2, flexdashboard)
  
Key Insights
- Tesla holds 42.9% of the EV market in Washington
- Model Y and Model 3 dominate Tesla registrations
- Compared Tesla vs competitors on electric range and MSRP
- Explored BEV vs PHEV adoption trends
- Identified top electric utilities serving Tesla ownners
